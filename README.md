# Introduction

This API provides functionality for Panorama site (https://github.com/jasrub/thesis-site)

# Running Locally

First, create a `config.js` file to configure the PostgreSQL URI. See `config.sample.js` for an example.

Then run:

```
npm install
npm start
```
